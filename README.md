# 🧩 Python node-app

A graphical Node Editor built with Python.

## 📌 Description

This project is a custom node-app system that allows creating and connecting nodes visually.
It is designed for learning purposes and experimenting with graphics programming and architecture.

## 🚀 Features

- Create nodes
- Connect nodes with edges
- Custom sockets
- Graphics view interaction
- Modular architecture

## 🛠 Technologies Used

- Python 3
- PySide
- OOP design principles
